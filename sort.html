Comando SORT

Descripción:

El comando sort muestra todos los archivos ordenados por la salida estándar. Se lee la entrada estándar, si no se define ningún ARCHIVO o bien se indica '-'

Opciones:

                
                    -u, --unique                     muestra los archivos sin contenido duplicado
                    -h, --human-numeric-sort         compara números en formato humanamete entendible (p.ej:2K,100, 45M,...)
                    -(opcion3), --(nombre completo)         (Descrioción de lo que hace la opcion3)

Ejemplos de uso:

                
                    sort ejemplo.txt       muestra ordenado alfaveticamente el contenido del archivo "ejemplo.txt"
                    sort -u ejemplo.txt    muestra ordenado alfaveticamente el contenido agrupando y eliminando el contenido repetido del archivo "ejemolo.txt"
                    sort -h ejemolo.txt    muestra ordenado numericamente en formato legible para el humano del archivo "ejemplo.txt"
