Comando SORT

Descripción:

El comando sort muestra todos los archivos ordenados por la salida estándar. Se lee la entrada estándar, si no se define ningún ARCHIVO o bien se indica '-'

Opciones:

                
                    -u, --unique                     (Descrioción de lo que hace la opcion1)
                    -h, --human-numeric-sort         (Descrioción de lo que hace la opcion2)
                    -(opcion3), --(nombre completo)         (Descrioción de lo que hace la opcion3)

Ejemplos de uso:

                
                    sort ejemplo.txt       muestra ordenado alfaveticamente el contenido, ejemplo.txt contiene "hola adios", 
                                           utilizando el comando, debolveria "adios hola"
                    sort -u ejemplo.txt    muestra ordenado alfaveticamente el contenido agrupando y eliminando el contenido repetido
                    sort -h ejemolo.txt    (Breve explicación de lo que hace el ejemplo)
